# 📘 Anotações - Bootcamp FAP 2026

---

## 🔹 Git e GitHub
- Commit salva alterações localmente
- Push envia para o repositório remoto (GitHub)
- Staging é a preparação das alterações antes do commit
- Controle de versão permite acompanhar evolução do projeto

---

## 🔹 HTML
- Estrutura da página web
- Uso de tags semânticas (header, main, section, footer)
- Organização do conteúdo

---

## 🔹 CSS
- Estilização da página
- Separação entre estrutura (HTML) e aparência (CSS)
- Layout básico e responsividade

---

## 🔹 JavaScript
- Linguagem responsável pela interatividade em páginas web
- Uso de variáveis e tipos de dados
- Estruturas condicionais (if/else)
- Funções
- Manipulação de eventos

---

## 🔹 Python
- Linguagem utilizada para lógica de programação e backend
- Variáveis e tipos de dados
- Estruturas de decisão e repetição
- Organização de código com funções

---

## 🔹 Banco de Dados (em andamento)
- Modelagem de dados (conceitual e lógico)
- Entidades, atributos e relacionamentos
- Introdução ao SQL (SELECT, INSERT, UPDATE, DELETE)

---

## 🔹 Documentação de Projetos (em andamento)
- Importância da documentação profissional
- README como apresentação do projeto
- Organização de repositórios
- Comunicação clara do que foi desenvolvido

---

## 💡 Insight importante
A organização, documentação e versionamento são tão importantes quanto o código para construir projetos profissionais.
